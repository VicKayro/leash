leash cloud beta

See what your Claude Code agents did last night — now across all your machines.

npx -y getleash connect

Free. No signup, no email — you get a private fleet URL. Metrics only: leash never uploads prompts, transcripts or file contents.

What you get — free

  • Fleet dashboard — every machine, one page: spend, sessions, night replay
  • Live platform checks — GitHub Actions, Vercel, Render, Railway, Cloudflare
  • Health at a glance — dead crons, zombies, runaway loops, missing budget guards

🐕 Watchdog

  • It barks — Discord ping the moment an agent dies, loops, or fails
  • npx getleash watchdog --discord <webhook> — armed in 30 seconds
  • Free, like everything else — open source, MIT

Local, no cloud

  • npx getleash — full fleet report in your terminal (nothing leaves the machine)
  • npx getleash watch — live monitor: cost ticking, burn rate per session
  • npx getleash guard --daily 25 — hard spend cap for Claude Code (nobody else has one)
